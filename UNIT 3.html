What is CSS?

  • CSS stands for Cascading Style Sheets. It is a style sheet language used for specifying the presentation and styling of a document written in a markup language such as HTML or XML. CSS allows for the separation of content and presentation, enabling the definition of layout, colors, fonts, and other visual characteristics of a web page. It is designed to improve content accessibility, provide flexibility in specifying presentation characteristics, and enable the sharing of formatting across multiple web pages. CSS can define various typographic characteristics, layout, and non-visual styles, and it plays a crucial role in creating visually appealing and responsive web pages

History of CSS
  • CSS, or Cascading Style Sheets, first emerged in the early 1990s as a solution to the growing complexity of web design. The web was initially text-based, and HTML was the primary tool for structuring web documents. However, as the demand for more visually appealing websites grew, the need for a dedicated styling language became apparent. CSS was first proposed by HÃ¥kon Wium Lie on October 10, 1994, while working with Tim Berners-Lee at CERN. It was not the only styling language in development at the time, but its cascading and developing sequence set it apart from the rest. The first version of CSS was invented in 1996, and CSS 2 was released in 1998, with work on CSS 3 beginning at that time. CSS 3 was very different from the other versions, as it was published as a set of separate documents known as modules, each dealing with a relatively small subset of the overall specification and either adding new features or refining and extending existing features.

Versions of CSS
  • The versions of CSS are as follows:
    (1). CSS1: This was the first version of CSS, officially released in 1996. It included properties for adding font properties, text attributes, and backgrounds. However, it faced a lack of dependable web browser support initially.
    (2). CSS2: The CSS2 specification was developed by the W3C and published as a recommendation in 1998. It added properties for positioning, allowed CSS to be used for page layout, and introduced styles for other media types.
    (3). CSS3: CSS3, the latest version, was introduced with the earliest drafts published in 1999. It adds presentation-style properties, allowing for the effective building of presentations from web documents. CSS3 is divided into several separate documents called modules.
    Additionally, there is also CSS 2.1, which is a revision of CSS2. It fixes errors in CSS2, removes poorly supported or not fully interoperable features, and adds already implemented browser features. CSS 2.1 went through a process of becoming a Candidate Recommendation and addressing problems with the patchy adoption of CSS and errata in the original specification.

CSS vs CSS3
  • CSS, which stands for Cascading Style Sheets, is a style sheet language used to describe the presentation of a document written in HTML or XML. On the other hand, CSS3 is the latest version of CSS, introducing several new features and enhancements to the styling capabilities of web pages.
    Some of the key differences between CSS and CSS3 include:
    - CSS provides basic styling properties such as color, layout, background, font, and border, while CSS3 introduces advanced features like animations, transitions, multiple backgrounds, gradients, and 3D transformations.
    - CSS3 supports responsive design, while CSS does not.
    - CSS3 can be broken down into modules, allowing for more modular and efficient development, whereas CSS cannot be split into modules.
    - CSS3 is faster than CSS and offers a wider range of color options, including HSL, RGBA, and gradient colors.
    In summary, CSS3 is an enhanced and more advanced version of CSS, offering a wider range of styling capabilities and improved efficiency for web development.

Types of CSS
  • There are three types of CSS: (1)Inline CSS (2) Internal or Embedded CSS (3) External CSS.
    (1). Inline CSS is a method of styling where CSS properties are directly applied to HTML elements within the same line of code.
    (2). Internal or Embedded CSS is defined in the head section of an HTML page, within a style element, and is used to define a style for a single HTML page.
    (3). External CSS is defined in a separate file and is used to define the style for multiple HTML pages. CSS3 is the latest version of CSS and introduced several new features and enhancements to the styling capabilities of web pages, including advanced animations and transitions, multiple backgrounds and gradients, multiple column layouts, opacity, rounded corners, flexbox, CSS grids, embedded fonts, responsive web design, and box shadows.

How to use in html document?

  • To use CSS in an HTML document, you can employ inline, internal, or external CSS.
    -Inline CSS: To apply a unique style to a single HTML element, use the style attribute of the HTML element. For example, `<h1 style="color:blue;">A Blue Heading</h1>` applies a blue color to the heading.
    - Internal CSS: Define a style for a single HTML page within the <head> section of the HTML page, using the <style> element. For instance, html <head> <style> body { background-color: linen; } h1 { color: maroon; margin-left: 40px; } </style> </head> defines the background color and heading style for the page.
    -External CSS: Create a separate CSS file containing style properties and link it to the HTML document using a link tag within the <head> section. For example, html <head> <link rel="stylesheet" href="styles.css"> </head> links the HTML document to an external CSS file named "styles.css". These methods provide flexibility in styling HTML documents, allowing for unique styles, consistent styles across multiple pages, and easier maintenance of the code.

    CSS Selectors and Their Types
    • CSS selectors are used to target HTML elements for styling. There are several types of CSS selectors, each serving a different purpose. The most common types include:
      (1). Element Type Selectors: These select elements based on their element name. For example, to select all paragraphs, the selector would be `p`.
      (2). ID Selectors: They select a specific element based on its unique ID attribute. The selector is preceded by a hash symbol (#). For example, to select an element with the ID "myElement", the selector would be `#myElement`.
      (3). Class Selectors: These select elements based on their class attribute. The selector is preceded by a period (.). For example, to select all elements with the class "myClass", the selector would be `.myClass`.
      (4). Universal Selector: It selects all HTML elements on the page. The selector is an asterisk (*).
      (5). Attribute Selectors: These select elements based on their attribute or attribute value. For example, to select all elements with a specific attribute, the selector would be `[attribute]`.
      (6). Pseudo-Class Selectors: They select elements based on a certain state, such as when a user hovers over an element or when it's the first child of its parent.
      (7). Pseudo-Element Selectors: These select and style a part of an element, such as the first line of a paragraph or the first letter of a heading.
      These are just a few examples of the many CSS selectors available, each offering a different way to target and style HTML elements.
